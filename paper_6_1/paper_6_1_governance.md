# Paper 6.1 — Governance Layer

This document defines the governance layer (M4) implemented in
`mapping_governance.py`.

## Purpose
Governance ensures that the operational substrate behaves safely,
predictably, and coherently.

## Governance Rules
A governance rule includes:

- Name
- Description
- Rule function

Rules are evaluated against runtime contexts.

## Governance Engine
The engine:

- Registers rules
- Evaluates rules
- Provides governance compliance reports

This layer stabilizes the substrate.
