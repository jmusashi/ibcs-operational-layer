\# Minimal Reference Simulation (MRS)

Aligned with Paper 6.1 — IBCS Operational Addendum



This module implements the Minimal Reference Simulation described in Section 6 of Paper 6.1.



\## Components

\- `mrs\_core.py` — Main simulation loop

\- `surface\_generator.py` — Heterogeneous decision surfaces

\- `reflexive\_engine.py` — Reflexive function R(Si, C)

\- `convergence\_monitor.py` — Serenpoint detection
